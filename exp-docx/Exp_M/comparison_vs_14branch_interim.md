# Exp_M vs 14分支 · 全分支多指标对比 + 配对 CI(交集题目,接近全量)

> 快照:Exp_M 跑到 **74 题**(q0–q73),14分支实验已全量 87 题。
> 交集(共 74 题),deepsearchqa 官方口径(`aggregate_ratings`,kimi-k2.5)。
> ⚠️ **n=74(接近全量),95% bootstrap CI:仍无任何对比显著(CI 都跨 0)**。**且新 run 的信号比 40 题时反而变弱**(见 §0),早先"mxdv 选层略好"的苗头已散——基本指向**主方案没拿到正向证据**。
> 历次:n=11(倒挂)→16(+2.3)→29(+7.9)→34(+7.7)→37(+6.9)→40(+6.5)→**74(+3.5,且 CI 跨 0)**。

## 0. 一句话结论(n=74,接近全量)⚠ 与早先快照不同
1. **新 run 的复用伤害反而缩小、且不显著**:full prefill 28.8 vs full reuse 25.4 = **+0.035**,CI **[−0.026, +0.091]**——从 40 题的 +0.065 一路缩到 +0.035,**CI 已明显跨 0**。
2. **同样 74 题,旧 run 的复用伤害却很清楚**:旧 prefill 33.5 vs reuse 24.1 = **+9.4 分**。**差异根源:新 run 的 full-prefill(28.8)显著低于旧 run(33.5)**——即**新 run 这批报告整体质量偏低、动态范围被压扁**(prefill 上界没拉开),信号自然难显。这是"不同 demo run"污染的一个实锤,也是新 run 信号弱的直接原因。
3. **mxdv 的"选层略好"苗头散了**:`mxdv50lay − mxdv50tok` 从早先 +0.038~0.049 掉到 **−0.001**;§1B 里 50% 行 mxdv−ΔV/ΔK 也都在 ±0.02 抖、符号不稳。**早先那个方向是噪声,不是信号。**
4. **总判**:到 74 题,**M×ΔV(主方案)没有任何正向证据**,连"复用伤害"在新 run 里都没显著;全量 80 题大概率仍是"方向弱/不显著"。

## 3. 关键发现汇总(n=74)
1. **复用伤害**:新 run +0.035(不显著)、旧 run 同题 +9.4 分(清楚)→ **不同 demo run 让新 run 上界塌了 ~5 分**,信号被压扁。
2. **M×ΔV 主方案**:相对纯 ΔV/ΔK(干净 50% 内部对比)全在 ±0.02、不显著、符号不稳 → **没拿到"注意力加权选位更优"的证据**。
3. **早先"选层 > 选 token"是噪声**:n≤40 看着像,n=74 归零(53−52=−0.001)。
4. **待全量 80 题最终定**,但趋势已较明朗:**主方案大概率 null。**
