## 最该看的一对：ICC（背景刻度）+ 1b（定真假）
- **ICC = 0.410**（σ²_间=0.6972 / σ²_内=1.0046）
- **1b 三数**：观测题内 R² 中位 = **0.0821**（adjusted 0.0263）；permutation null 中位 = **0.0318**（95分位 0.0501）；**p = 0.000**。
  - 题内相关符号：正 31 / 负 27 题；超噪声 R² = 0.0503；留给 M 的空间 ≈ **0.561**。
  - **判读**：观测显著超噪声底(p=0.000)**但幅度极小**(超噪声 R²=0.050)→ 几何题内信号真实但只解释一小条,M 空间仍大(~0.56)→ **M×Δ 值得接**(几何当辅、M 补主)

## 综合判读（1b 定真假、ICC 定背景刻度）
- → 几何题内信号真实但极小(超噪声 R²=0.050),M 空间仍大(~0.56) → **M×Δ 值得接**(几何辅、M 主)
