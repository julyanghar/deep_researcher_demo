# Exp A 全量(58 题)结果 —— 几何对决策扰动只有**弱**信号,M 是主缺口

> 模型 Qwen3-32B,vLLM+LMCache。58/60 题(q11/q22 demo 在超难多条件题上崩,非测量 bug),
> 1006 个 swap 事件,131 次决策翻转。RoPE 前缀核对 OK(bf16 地板、位置对齐正常)。
> 数据:[regress.json](regress.json) · 方法:[../IMPLEMENTATION.md](../IMPLEMENTATION.md)
> **注:初版结论是"干净 NULL/几何没用",经 |Δz| 再分析已修正(见下「⚠️ 修正」节)——
> 改用不饱和的段级靶子 |Δz| 后,几何浮现出弱但一致的信号;但仍弱到不足以单独 gate。**

## 一句话
用**饱和**的 action_KL 测时,几何相关 ≤0.08、题内 43% 正(像没信号);但 action_KL=½·边界性·(Δz)² 把段级信号压扁了。**换成不饱和的段级靶子 |Δz| 后,几何浮现弱但一致的信号**(题内 **59% 正**、Spearman ~0.13)。**结论:几何对决策扰动有弱真信号(之前"NULL"高估),但幅度只解释一小部分 → 单独不足以 gate,需 M×Δ。**

## 详细方法:统计了哪些数据、相关性怎么算(大白话)

### 第 0 步:什么是一个"事件"(数据的基本单位)
基本单位是一个 **"段-swap 事件"**:在某道题、最后一轮的 supervisor 决策上下文里,**只把其中一个 summary 段 s_i 换成复用值 KV^r、其余全保持真值 KV\*,看决策怎么变**。
- 58 题,每题最后一轮上下文平均 ~17 个 summary 段 → 共 **1006 个事件**。
  > **为什么是 ~17 段而非 9 段**:agent 是三层扇出 `max_iterations(3) × max_followups(3) × max_queries_per_researcher(3)`,**一个 summary 段 = 一个子查询的压缩结果**,每轮 ≤9 段、每题 ≤27 段。完整工作流(每轮几个 summary、段=什么)见 [deep_researcher_demo 工作流说明](../../../../deep_researcher_demo/docs/workflow.md)。
- 每个事件记两组数:**X = 这段 KV 偏真值多少(几何)**,**Y = 换了这段决策变了多少**。相关性就是问:**X 大的事件,Y 是不是也大?**

### 第 1 步:X = 几何(复用 KV 偏真值多少)
每个事件,从 truth 那遍落盘的 KV dump 里取这段 s_i 的两份 KV:
- **KV\***(真值):在 supervisor 上下文里**重新算**这段的 K、V(全重算 = full prefill)。
- **KV^r**(复用值):这段当初 worker 算好、被 LMCache 存下,复用时**载入 + RoPE 旋转**到 supervisor 的位置。

两份形状都是 `[K或V, 64层, 段token数, 8个KV头, 每头128维]`。对每个 (K或V, 层, token, 头) 算三种偏差:
- **方向 cos** = `1 − 余弦相似度(KV^r, KV*)`:方向转了多少(0=同向,1=正交)。
- **幅度 norm** = `|‖KV^r‖ − ‖KV*‖| / ‖KV*‖`:长度变了多少(只看模长)。
- **欧氏 eucl** = `‖KV^r − KV*‖`:直线距离(方向+幅度一起)。

得到一堆逐层逐token逐头的偏差,再**聚合成标量**(相关性要一段一个数):对 K、V 各自 × 三种度量各自,算 5 个聚合 —— 整体均值、整体最大、浅层均值(前1/3层)、中层均值、深层均值。
→ 共 **3 度量 × 2(K/V) × 5 聚合 = 30 个几何特征**(如 `V_eucl_deep`=V欧氏偏差的深层均值、`K_norm_max`=K幅度偏差最大值)。
**排除分隔符**:段按 `<|fim_pad|>` 切,只取段内内容 token(分隔符本身漂最狠、会当离群,不计入)。

### 第 2 步:Y = 决策变化(换了这段,决策推偏多少)
同一道题跑两遍 supervisor 自由生成决策(temp=0):
- **truth 遍**(全段真值)→ 决策 JSON;在 `"status": "` 后那一位,读 **continue(token 9534)** 与 **complete(token 14737)** 的 logprob → 归一化成两点概率 `(p*_cont, p*_comp)` = **p\***。
- **swap-s_i 遍**(只这段复用)→ 同样读出 **p^r**。
- **action_KL** = 两个两点分布的 KL:
  `action_KL = p*_cont·ln(p*_cont/p^r_cont) + p*_comp·ln(p*_comp/p^r_comp)`
  大白话:**换这段 KV,把"continue vs complete"的概率推偏了多少**(0=没动,越大越偏)。
- **flip**:两遍 argmax 是否翻转(continue↔complete)= 决策真翻没翻。
- **followup_jaccard**(只 continue 时):两遍生成的 followup 各拿去搜,比 URL 集合的 Jaccard 差。

### 第 3 步:算相关性(30 个几何特征 各自 vs action_KL)
把 1006 个事件排成两列(某几何特征一列、action_KL 一列),算 **Pearson**:
`Pearson = Σ(x−x̄)(y−ȳ) / [√Σ(x−x̄)² · √Σ(y−ȳ)²]`
大白话:**几何大的事件,action_KL 是不是也大?** +1=完全同涨跌,0=毫无关系,−1=完全反向;`Pearson²` ≈ 几何能解释 action_KL 变化的百分比。
再算 **Spearman**(两列各换成"名次"再 Pearson)= 捕捉单调但非线性的关系。30 个特征都算,按 |Pearson| 排序,最强那个 = "Δ 候选"。
**结果**:最强的 `V_norm_deep` 仅 Pearson=0.076(Pearson²<0.6%)→ 几乎没关系。

### 第 4 步:三重稳健性检查(确认 NULL 是真的,非算法/混池伪影)
1. **混池相关**(上一步):1006 事件直接算 → 最强 0.076,弱。
2. **题内相关**(防"跨题尺度差异冲淡"):在**每道题内部**,它那 ~17 段的几何 vs action_KL 算一次 Pearson,再对 58 题取平均 → 均 +0.017、中位 −0.04、仅 43% 为正。**题内本身就没关系** → 不是混池造的假 NULL。
3. **翻转判别**(换个问法):131 个真翻转事件 vs 不翻转,比各几何特征均值 → 1.01–1.03×,几乎一样 → 几何分不出"哪些会翻"。
4. **join 验证**(防错位):抽查每段的几何 token 段 == 它对应事件的 swap token 段(同 seg_index、同 (start,end))→ 一致 → 不是"几何配错事件"导致的假 NULL。

### 数字总账
58 题 · 1006 事件 · 每事件 30 几何特征 + action_KL/flip(其中 321 事件有 followup_jaccard)· 131 翻转 · 全部行 → [regress.json](regress.json)。

## 证据(三重,互相印证,排除了方法学伪影)
1. **混池相关弱**(线性 Pearson + 单调 Spearman 都弱):1006 事件,30 个特征:
   - **action_KL**:最强 Pearson=0.076(`V_norm_deep`);最强 **Spearman=0.164**(`K_cos_mid`,Spearman²≈2.7%)。
   - **followup_jaccard**:最强 Pearson=0.096、Spearman=0.110(`K_eucl_max`)。
   - Spearman 略高于 Pearson(说明就算放宽到"单调非线性",信号也只到 ~0.16),且最高的几个都是 **K 中层**——硬要挑"最不无用"的量是 K 中层方向漂移,但 <3% 解释力,不可用。n=1006 下统计显著但**效应可忽略**。
2. **题内相关≈0**(控制题级尺度差异;== 抛硬币):每道题内部单独算它 ~17 段的 几何 vs action_KL 的 Pearson → 58 个数:**正 25 / 负 33(43% 正)、均 +0.017、中位 −0.04**,对称散在 −0.73~+0.69(11 题强负、16 题强正,两边对冲)。**若几何真预测决策,应绝大多数题为正(>80%);43% ≈ 抛硬币 = 题内本身就没关系**,排除"跨题混池冲淡真信号"。
3. **几何判别不了翻转**:131 个翻转事件 vs 不翻转,各几何量均值比 **1.01–1.03×**(几乎相同)。

## action_KL 分布(为什么"换单段"多半不动决策)
中位 **0.0077**、均 0.118、p90 0.27、max 5.98;**25% 近 0、仅 6% >0.5、1% >2**。
→ 一个上下文里有 ~17 段,**换其中一段,多数情况决策纹丝不动**;只有少数(决策本就borderline + 那段恰好关键)才动,而这"少数"也不由几何大小决定。

## 怎么解读(3 条,不互斥)
1. **缺了注意力 M(最可能,呼应计划的 M×Δ 假设)**:几何只说"复用的 KV 有多偏",但没说"决策有没有去看这一段"。一段偏得再多,决策不 attend 它就不影响 → **几何单独(Δ)注定弱,要乘上 M(决策对该段的入向注意力)**。
2. **决策对单段复用本就稳健**:边际影响小(中位 KL 0.008),所以信号稀疏。
3. **2-way KL 会饱和**:action_KL 被"决策有多 borderline"(题/轮级属性)主导,而非段级几何;自信的决策怎么扰都不动,borderline 的一碰就翻。

## ⚠️ 修正(|Δz| 再分析)—— "干净 NULL"说重了
原分析拿几何关联 **action_KL**,但 2-way KL 在中心可二阶展开:
`KL(p*‖p^r) ≈ ½·p*(1−p*)·(Δz)²` = **决策边界性 × 扰动²**。边界性 `p*(1−p*)`(题/轮级)把段级信号冲淡,且 (Δz)² 在单段 regime 把信号压 16–289×(累积复用动 ~17 项、Δz 大 √17~17 倍)。
**正确的段级靶子是 |Δz|**(logit gap 扰动本身),且零重跑——`Δz=(logp_cont−logp_comp)_swap−(...)_truth`,由已存的四个 logprob 直接算。重做后:
- **|Δz| 不退化**:中位 0.75(vs action_KL 0.008),仅 8% 近 0(vs 25%)。
- **池化** 几何→|Δz| 最强 Spearman 0.126(同量级)。
- **题内方向翻转**:从 action_KL 的 43% 正(抛硬币)→ **|Δz| 的 59% 正、中位 +0.066(一致弱正)**。
→ **KL 饱和确实掩盖了一个真实的弱段级信号,"几何没用/干净 NULL"的说法收回。** 但信号**仍弱**(~0.13、题内 +0.07),几何幅度只解释扰动的一小部分 → **M 仍是主缺口**,M×Δ(分支 A)该接、且用 **|Δz| 当靶子**。

## 对 gate 的直接含义
**只靠几何 Δ 来 gate 复用,行不通**(预测不了决策变化)。gate 必须引入**注意力权重 M**,或换信号。
