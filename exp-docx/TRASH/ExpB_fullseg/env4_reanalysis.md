# env4 零成本重分析 —— 换掉"被异质性坑了的平均 F1 差"

> **结论先行**:把统计量从"逐题 F1 差的平均"换成**符号检验 + 分层 + 分布**后,**"复用让答案变差"在现有 40 题上站住了(borderline 显著)** —— 之前 Set-only 平均 0.056「不显著」,是被**持平题稀释 + 反向题放大方差**坑了,不是真无害。**不加样本、不跑新实验,只换统计量。**

## 为什么要换
这 40 题极不同质:**~19 题持平**(F1差≈0)、**几题重创**(掉 1.0)、**1 题反向**(q27,reuse 反而 +0.8)。把它们一锅平均 → 持平题把信号稀释、反向题把方差撑大 → **CI 跨 0**。这不是"复用无害",而是"均值是错的统计量"。

先看一个直接证据:**全 40 题配对 F1 差 = 0.151 [95%CI 0.041, 0.272],CI 不跨 0 → 显著**。Set-only 的 0.056「不显著」是拆分 + 持平稀释的产物。

---

## 分析1:符号检验(数方向,不管幅度)
问"变差的题是不是显著多于变好的题"(持平题剔除、不稀释;反向题不放大方差)。二项双侧 p。

| 组 | 阈值0.05:变差/变好/持平 | p | 阈值0.10 | p |
|---|---|---|---|---|
| **全 on-topic(n=31)** | **10 / 2 / 19** | **0.039 ✓** | 8 / 1 / 22 | **0.039 ✓** |
| Set(n=21) | 7 / 2 / 12 | 0.18 ✗ | 5 / 1 / 15 | 0.22 ✗ |
| Single(n=10) | 3 / 0 / 7 | 0.25 ✗ | 3 / 0 / 7 | 0.25 ✗ |

- **全 on-topic 显著**(p≈0.039):复用让**变差的题(10)远多于变好的题(2)**。
- 阈值敏感度:0.05/0.10 显著,**0.01 时 p=0.092 临界**(说明结论稳健性中等,别只信一个阈值)。
- Set / Single **单拆都不显著**(n 太小)→ **显著性来自合并**。

## 分析2:按"过程偏没偏离"分层(最强一环)
用 env2(status)或 env3(followup)给每题打标"复用是否真搅乱了过程",分两层:

| 层 | n | F1 差 [95%CI] | 符号检验 |
|---|---|---|---|
| **偏离层**(复用真搅乱了过程)| 20 | **0.241 [0.086, 0.426]** ✓ | 变差8/变好1,p=0.039 |
| **未偏离层(阴性对照)** | 11 | **−0.058 [−0.213, 0.032]** ≈0 | 变差2/变好1,p=1.0 |

- **过程被搅乱时,F1 掉 0.24(显著);没搅乱时,F1 差≈0(甚至略负)。** → **"过程被搅乱 → 答案变差"因果链成立**,未偏离层≈0 是漂亮的**阴性对照**(证明 F1 掉是跟着"过程偏离"走的,不是无差别噪声)。
- ⚠ **选择偏差**:用"过程偏离"筛题、而过程偏离与答案变差可能同因 → 偏离层数会偏高。**此数不能单独当因果证据**,但配分析1(无筛选、也显著)一起,就站得住。

## 分析3:F1 差的分布(看形状)
on-topic 31 题:**变差 10 / 持平 19 / 变好 2**。

```
 +1.0  ### 3        ← 重创(全在偏离层)
 +0.8  # 1
 +0.6  # 1
 +0.1  ### 3
  0.0  #################### 20   ← 一大堆持平
 -0.0  ## 2
 -0.8  # 1          ← 反向 q27
```

- **重创题(F1差>0.5)= q13, q29, q3(都 1.0)、q32(0.8)、q22(0.67)—— 全部是"过程偏离"题。**
- **反向题 q27(reuse +0.8)**:Set 题,prefill 没搜到街道名、reuse 碰巧搜到了。**"换条路撞了运气",单题方差、非系统优势,也非 bug。**
- 这个形状(大堆≈0 + 正向尾巴 + 1 个负值)正好解释了**为什么均值小且 CI 跨 0**:被持平题拉低、被 q27 撑大方差。**形状比均值诚实。**

## 分析4:Single −0.30 的脆性(留一法)
- Single 的 0.30 由 **q13、q29、q3**(都 1→0)撑起。
- 留一法:剔任一题后均值落在 **0.222 ~ 0.333** → 剔最狠的 q13 也还有 0.222。**3 题撑、但不是 1 题独撑,中等稳健**;n=10 仍小,别当铁证。

## 综合判读
| 分析1(符号检验)| 分析2(分层)| → 结论 |
|---|---|---|
| 全 on-topic **显著**(p≈0.039,阈值0.05/0.10)| 偏离层 −0.24 显著、**未偏离层≈0(阴性对照)** | **"复用有害"站住(borderline)**——均值坑了,换合适统计量即过 |

**最终**:
- **"复用让答案变差"在 40 题上 borderline 成立**(全 on-topic 符号检验 p≈0.039;过程偏离层 F1 −0.24 显著且有阴性对照)。**之前的"不显著"是均值这个错指标 + 拆分 + 持平稀释造成的。**
- **但仍是 borderline,不是铁证**:① 只在**合并 on-topic** 显著(Set/Single 单拆不显著);② p 刚过 0.05、**阈值 0.01 时不显著**;③ 分层有选择偏差。
- **机制链更完整了**:过程偏离(env2/env3)→ 答案变差(分层 −0.24 + 阴性对照),且**所有重创题都在偏离层**。
- **go/no-go**:从"定量没站稳"上修到"**borderline 站住、机制清楚**"。要变铁证,加样本到 ~60–100 题最直接(让符号检验/分层的 p 远离 0.05)。
