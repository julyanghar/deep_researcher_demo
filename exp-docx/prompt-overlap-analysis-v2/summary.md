# online100 生成文段 ↔ 输入 prompt 逐字重复统计

- 样本：1912 条生成调用（/home/yilin/deep_researcher_demo/eval/results/drbench/online100_v2）
- 字符级；prompt = system+user 拼接；SAM 求最长逐字匹配。
- **Coverage@k** = 生成文本被「长度≥k 的逐字片段」覆盖的字符占比；headline = Coverage@8。
- raw = 原样；prose = 生成文本先去 URL/markdown 标记再比（分母=剥离后长度）。
